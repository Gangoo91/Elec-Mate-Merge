# OpenAI Models Reference

## Available Models (as of 2025)

### GPT-5 Mini
- **Model ID**: `gpt-5-mini-2025-08-07`
- **Use case**: Fast, efficient responses
- **Parameter**: `max_completion_tokens` (NOT `max_tokens`)
- **Notes**: This model exists and works! Don't replace with gpt-4o-mini.

### GPT-4o Mini
- **Model ID**: `gpt-4o-mini`
- **Use case**: Legacy/fallback
- **Parameter**: `max_tokens`

## Important Notes
- GPT-5 mini was released in August 2025
- Always use `max_completion_tokens` for GPT-5 models
- Always use `max_tokens` for GPT-4 models
